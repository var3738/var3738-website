# VAR 37/38 Platform

Institutionalizing Youth Power: From the Streets to the Ballot.

A premium "Civic-Tech" ecosystem built for the **VAR 37-38** movement. This platform transforms youthful energy into structured civic blocks through technology, dialogue, and institutional engagement.

## Visual Identity
- **Aesthetic**: Modern Dark-Mode SaaS (inspired by thirdweb.studio).
- **Primary Color**: VAR Crimson Red (`#D0171D`).
- **Core Design**: Deep charcoal/black backgrounds (#0A0A0A) with clean borders (#262626), glassmorphism effects, and premium typography (Inter/Geist).

## Key Features
- **Democracy Activated**: A 3-Stage Pilot framework for ward-level youth mobilization.
- **Impact Dashboard**: Real-time analytics on movement health, voter density, and mobilization funnels.
- **Secure Registration**: Multi-step, encrypted registration flow for youth champions.
- **Field Reports**: Ward-level granular data and governance intelligence from Trans Nzoia.
- **Premium Lightbox**: High-fidelity media preview for capturing the spirit of the movement.

## Technology Stack
- **Framework**: Next.js 16+ (App Router, Turbopack)
- **Styling**: Tailwind CSS v4.0
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Infrastructure**: Powered by Uamuzi Foundation

## Project Structure
- `/app`: Main application routes (Home, Democracy Activated, Tech, Impact, Reports, etc.)
- `/components`: Reusable UI components adhering to the Dark SaaS design system.
- `/public`: High-resolution imagery from the Trans Nzoia Townhall series.
- `/website-content`: Source PDF documents for authoritative organizational content.
